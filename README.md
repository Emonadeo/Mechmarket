## Quickstart

### Install dependencies

Make sure you have [Node.js](https://nodejs.org/) and NPM installed. *(NPM is usually included in your Node.js installation)*
```
npm install
```

### Serve locally

Serve the app with hot reloading locally on http://localhost:8080.
```
npm start
```
